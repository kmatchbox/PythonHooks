# Save Batchghroups to a location

Saves all the selected backgroups on the desktop and saves them to a location specified by the user. Useful if you need to share all your batchgroups with someone or, for whatever reason, you might want them all in one easily accessed location.

### How To Use:
1. Select batchgroup(s) on the desktop
2. Right-click > Archiving Prep > Save batchgroup to ...
3. Select destination
